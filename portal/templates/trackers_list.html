How to add a new tracker

30 seconds — no code change to routes/templates needed

  1. Drop the CSV file into portal/trackers/ (lowercase filename, no spaces).
  2. Open portal/lib/trackers.py and add an entry to TRACKERS: 
    "alpha": {
    "id": "alpha", "title": "Alpha Tracker", "emoji": "⚡",
    "csv": "alpha_tracker.csv", "product": "Alpha",
    "creator_col": "Creator", "poc_col": "POC",
    "skip_rows": 0,  # set to 1 if the first row is totals
    "description": "Alpha campaign targets & achievement.",
},
  3. Reload the portal — the new tracker appears here and at /trackers/<id>.

The system auto-detects all the CSV's columns, filters by POC for the logged-in POC, and adds live DB columns (30d GMV, last post, status). No template work needed for new trackers — they all share this view.
