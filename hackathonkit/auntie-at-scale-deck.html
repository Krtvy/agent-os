Operator opens Whisk + Kling + ElevenLabs + CapCut. Generates one reel in 4 hours. Proves quality.

Phase 2 — Semi-automated (Week 2)

n8n / LangGraph orchestrates Whisk → Kling → ElevenLabs → ffmpeg edit. Operator reviews + approves. 5 reels/day per operator.

Phase 3 — Full pipeline (Month 2)

JSON brief (brand + product + persona) → automated pipeline → produces 30-reel season. Human approves 3 checkpoints: character fidelity, brand safety, batch publish.

50 operators × 5 brands × 30 reels = 7,500 reels/month. Auto-schedule via Meta Graph API. Virality Predictor scores reels pre-publish; only top 70% goes live.
