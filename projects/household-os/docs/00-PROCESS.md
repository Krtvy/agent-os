# Household OS — Development Process

## Document Order (do not skip steps)

```
STEP 1  RESEARCH.md        ← Indian household data, papers, stats, datasets
STEP 2  PRD.md             ← Product Requirements Document
STEP 3  TRD.md             ← Technical Requirements Document  
STEP 4  ERD.md             ← Entity Relationship Document (data models)
STEP 5  API-SPEC.md        ← API specification
STEP 6  DATASET-PLAN.md    ← What data we collect, build, or source
STEP 7  BUILD.md           ← Phase-by-phase build plan
```

## Rule
No code is written until STEP 1-4 are complete.
No API is designed until STEP 2 is complete.
No dataset is built until STEP 1 is complete.
